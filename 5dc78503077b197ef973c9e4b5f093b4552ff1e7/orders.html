Search and view orders

  • Click the Order icon to open the Orders page to view existing orders that are assigned to you.
    • Search for orders by ID, BAN, or Subscripton ID.
    • Click the advanced search icon to use the advanced search box.
  • From the Home page, click any of the tabs under Orders, such as Assigned.

Orders screen Actions

On the Orders screen, you can do the following:

  • View orders by status by clicking the Status dropdown menu for which the default option is All.
  • Use the filter and layout icons ( to filter the displayed fields.

    Note: You can only edit an order that has a Pending Review status.
