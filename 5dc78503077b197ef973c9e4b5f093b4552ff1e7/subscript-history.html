Subscription History

View the history of a subscription.

  1. Click Subscriptions on the left navigation, and on any subscription card for Prepaid or Postpaid, click the ellipsis (…) on the top right and select Subscription History.
  2. Click any of the following tabs to view a detailed history:

    Click the results filter icon to filter the fields that are displayed for each tab.

    • Balance History
      • Click the date picker icon in the search box to input a date range for the search results and enter a number in the Limit field to to limit the number of results that are displayed if you don't want all results.
      • Click the eye icon () to display results in a list.
    • Bucket History
      • Click the date picker icon in the search box to input a date range for the search results and enter a number in the Limit field to to limit the number of results that are displayed if you don't want all results.
      • Click the eye icon () to display results in a list.
    • Charging History
      • Click the eye icon () to display results in a list.
    • Provisioning History
      • Click the eye icon () to display results in a list.
    • Transaction History
      • Click the date picker icon in the search box to input a date range for the search results and enter a number in the Limit field to to limit the number of results that are displayed if you don't want all results.
      • Click the eye icon () to display results in a list.
