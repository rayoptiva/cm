Creating a Party

Create a Party (Individual) to use as basis for creating a customer

Create the Party (Individual) to record an individual before the assignment of the customer role.

  • From the Dashboard click + Create > New Party
  • From the left navigation bar, click Party > + Create > Individual Party

Click the following sections in the right pane, add relevant details and click Save to save as draft or click Submit to create the party assuming that the required information is input.

  • Individual [Required]
  • Contacts
  • Individual Identification

    A party can't be validated until the Identification is verified.

    Note: Only a validated Party can be used for customer creation.
  • Credit Profile
  • Tax Exemption Certificate
  • Additional Information
    In the Additional Information section, you add the following specific information.
    • Language Ability
    • Skill
    • Disability
    • External Reference
    • Additional Information
    When you complete the required sections (Individual), the Submit button is enabled, which enables you to:
  • Submit the data by clicking Submit.
    • View the summary, Edit and Update the record.
    • Confirm the data by clicking Confirm to create the Party.
Note: At any time during the data entry process you can click Cancel to Save as Draft or Delete to delete the record.

Individual

Add the Individual profile.

This is a required section where Given Name and Family Name are required fields.

  1. Complete the Basic Information and Personal Information sections.
  2. [Optional] Add another name as required.
  3. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can save, complete the remaining sections and then submit to create the Party.
      • You can click Save > Submit to create the Party when the required fields/sections are completed.

        The save option only saves your current input and does not create a Party.

    • Click Submit to create the Party.
      • View the Party Summary and edit/update sections as required.
      • Click Edit, make changes and then click Update to save changes.
      • Click Confirm to confirm the data and the Party creation.
    • Click Cancel to Delete the record, or to Save as Draft.
Note: As this is the only required section, you can submit this record, review the summary and click Confirm to confirm the data that you've entered and the Party is created.

When you submit without validating by completing the Individual Identification section, the Party is in an Initialized state but not a Validated state.

Contacts

Add a contact for this Party.

Add the contact medium characteristics that can be used to contact a party.

  1. Click Contacts .
  2. Click + Add Contact or Add to add a new contact.
  3. Select Medium Type.

    The type of the contact medium, such as, email address, telephone number, or postal address.

    If this is this first entry for this Medium Type, by default, it is marked as Preferred in when the party is created. Any subsequent entries for this type will not be marked as Preferred.

  4. Select Contact Type.

    The type of contact, for example, phone number such as mobile, fixed home, or fixed office.

  5. Add address information and dates for this contact.
  6. [Optional] Select Preferred if you want to make this option the preferred contact option.
  7. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can save, complete the remaining sections and then submit to create the Party.
      • You can click Save > Submit to create the Party when all the required fields/sections are completed.

        The save option only saves your current input and does not create a Party.

    • Click Submit to create the Party.
      • View the summary and edit/update sections if required.
      • Click Edit, make changes and then click Update to save changes.
      • Click Confirm to confirm the data and create the Party.

        You are not requied to click Save before you click Submit as the Submit action submits any data that is entered independently of the Save action, which saves your work locally.

    • Click Cancel to Save as Draft or to Delete the record.

As the required section (Individual) is completed, you can submit this record, review the summary and click Confirm to confirm the data that you've entered and the Party is created.

Individual Identification

Add identification for the individual.

  1. Click Individual Identification.
  2. Click + Add Identification.
  3. Add identification information.
  4. Upload any supporting identification files.
  5. Select Validated to validate this party.
    • You can Mark as Validated at a later stage by selecting Mark as Validated under the Actions menu of the search results screen or when you edit the Party.
      Note: To be validated, a Party much have identification added that is validated.
  6. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can save, complete the remaining sections and then click Submit to create the Party.
      • You can click Save > Submit to create the Party.

        The save option only saves your current input and does not create a Party.

    • Click Submit to create the Party.
      • View the summary and edit/update sections if required.
      • Click Edit, make changes and then click Update to save changes.
      • Click Confirm to confirm the data and create the Party.
    • Click Cancel to Save as Draft or to Delete the record.

As the required section (Individual) is completed, you can submit this record, review the summary, edit if required, and click Confirm to confirm the data that you've entered and the Party is created.

Credit Profile

Add the Credit profile.

  1. Click Credit Profile.
  2. Click Add.
  3. Add Credit Agency Type.

    These are predefined levels:

    • No credit rating required
    • Manual
    • External Agency
  4. Select Credit Agency Name.
    • If you select Experian or another external agency for the Credit Agency Name, you click Get Score to get the credit rating from an external credit rating agency and the fields are filled with relevant credit rating data.
    • For A manual credit rating, you must input the data and add the Add Valid from and Valid until dates.
    • For no rating required, you don't need to add any credit rating data
  5. Add Valid from and Valid until dates.
  6. Save, Submit or Cancel.
    • Click Save to save your current input as a draft, which does not create a Party.
      • You can save, complete the remaining sections and then submit to create the Party.
      • You can click Save > Submit to create the Party.

        The save option only saves your current input and does not create a Party.

    • Click Submit to initiate the Party creation.
      • View the summary and edit/update sections if required.
      • Click Edit, make changes and then click Update to save changes.
      • Click Confirm to confirm the data and create the Party.
    • Click Cancel to Save as Draft or to Delete the record.

As the required section (Individual) is completed, you can submit this record, review the summary and click Confirm to confirm the data that you've entered and the Party is created.
