Closed User Groups

A closed user group (CUG) is a group configuration that limits access outside of a defined set of group members. Users in the same group receive discounted rates when they call one another.

For example, if all marketing department employees belong to a CUG, they receive discounted rates when calling other marketing group members.

CUGs support short codes. If a CUG is short-code enabled, you can assign dialling codes to each member so that other members can call them quickly by using a shortcut number.

Private and Public CUGs

Private CUG

A private CUG is owned by the customer/account and the associated party decides who can join.

Public CUG

A public CUG is owned by the operator and not by the customer/account. The subscriber can only join an existing public CUG and abide by the rules to that CUG.

Note: Subscriptions can belong to more than one CUG at a time.

Viewing closed user groups

Go to the Closed User Groups (CUG) screen.

  1. Click Subscriptions on the left navigation panel.
  2. Select a subscription or search for a subscription.
  3. Click the ellipis (…) to open the menu and select Closed User Groups.

Editing closed user groups

On the Closed User Group screen, you can view a list of Private and Public CUGs for this subscription.

Private CUG

For Private Closed User Groups, you can view, edit, or delete a private CUG.

  • Click the eye icon () to view members.
  • Click the edit icon () to edit the group.
    • Add new members by clicking + Add New
    • Delete to remove a member from the group.
    • Edit member details.
  • Click Update when you make changes to commit your changes.

Public CUG

For Public Closed User Groups, you can join an existing public CUG or remove the subscriber account from the group.
