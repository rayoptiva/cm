Subscription History

View the history of a subscription.

To view subscription histories:

  1. Click Subscriptions on the left navigation.

    2. Select Subscription History.
  2. Click any of the following tabs to view a detailed history:

    Click the results filter icon to filter the fields that are displayed for each tab.

    • Balance History
      • Click the date picker icon in the search box to input a date range for the search results and enter a number in the Limit field to to limit the number of results that are displayed if you don't want to view all the results.
